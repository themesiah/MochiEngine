Detailed Description

This AnimationFactory takes Aseprites spritesheet export format and automatically generates all needed animation data for that spritesheet.

Animation data is expected as a JSON with specific configuration:

Sheet Type: Packed

Split Layers: False

Split Tags: False

Array, Meta Tags

Item Filename: {tag}_{tagframe}

Item Tagname: {tag}

Check AsepriteExport.png
